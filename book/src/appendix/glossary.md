# Glossary

**APR** - Aprender Pattern Repository file format (`.apr`)

**CITL** - Compiler-in-the-Loop methodology

**Coreutils** - GNU core utilities

**Decy** - C→Rust transpiler

**Depyler** - Python→Rust transpiler

**Oracle** - Pattern database for fixing transpilation errors

**P0/P1/P2** - Priority tiers based on complexity
