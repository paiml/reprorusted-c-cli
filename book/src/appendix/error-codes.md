# Rustc Error Codes

## E0506: Cannot assign to borrowed (31%)
Attempting to assign to a value while it is borrowed.

## E0499: Multiple mutable borrows (21%)
Creating multiple mutable references to the same data.

## E0382: Use after move (16%)
Using a value after ownership has been transferred.

## E0308: Type mismatch (10%)
Types don't match in assignment or function call.

## E0133: Unsafe required (7%)
Operation requires unsafe block.

## E0597: Does not live long enough (6%)
Reference outlives the borrowed value.

## E0515: Cannot return reference (5%)
Returning reference to local variable.
