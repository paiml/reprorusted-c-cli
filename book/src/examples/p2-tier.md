# P2 Tier (Advanced)

The P2 tier contains 5 high-complexity utilities.

## sort (1500 LOC)
**Primary Errors**: E0499, E0506 - Complex sorting with callbacks.

## uniq (250 LOC)
**Primary Error**: E0382 - Streaming deduplication.

## chmod (300 LOC)
**Primary Errors**: E0308, E0133 - Bitwise operations with syscalls.

## chown (250 LOC)
**Primary Error**: E0133 - Privilege operations with unsafe syscalls.

## cut (400 LOC)
**Primary Errors**: E0382, E0597 - Field parsing with lifetime constraints.
